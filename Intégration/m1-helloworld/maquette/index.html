Au cours de cette formation nous allons voir 2 matières:

L'intégration

Le HTML:


Le HTML sert à créer le contenu du site: elle va contenir uniquement la partie textuelle du site

Le CSS


Le CSS va nous permettre d'ajouter la couche graphique à notre site: les couleurs, les formes, les fond, etc.

Le développement

Le Javascript:



Javascript est un langage de programmation de script employé pour rendre une page interactive, le comportement de la page et son apparence pourra varier en fonction de certain paramètres.

Le PHP:




Le PHP est un langage de programmation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire remplis par l'utilisateur etc).

MySQL:



My SQL est un système de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple: formulaire de création de compte) pour ensuite être ré-utilisé au moment approprié (ex: connexion à un site internet avec identifiant et mot de passe).
